### clouds game

#### Rules
* 0 = safe
* 1 = to be avoided
* The player can jump to a cloud => 2

Input: (array) an array of binary integers
Output: (int) min number of jumps required


INPUT               |   OUTPUT
2, [0,0]            | 1
3, [0,1,0]          | 1
4, [0,1,0,0]        | 2
5, [0,1,0,1,0]      | 2
6, [0,0,0,0,1,0]    | 3
7, [0,1,0,0,0,1,0]  | 4
